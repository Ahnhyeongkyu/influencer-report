## 핵심 요구사항

### 1. 5개 플랫폼 통합 크롤링
- 샤오홍슈 (小红书)
- 페이스북
- 인스타그램
- 유튜브
- 디카드 (대만 플랫폼)

### 2. 웹 인터페이스 (Streamlit)
- 캠페인명 입력
- 광고주명 입력
- 인플루언서 포스팅 URL 리스트 입력
- 내부 사용자 전용 (인증 필요)

### 3. 수집 데이터
- 좋아요 수
- 즐겨찾기/저장 수
- 댓글 수
- 공유 수
- 조회수
- 작성자 정보

### 4. 분석 및 시각화
- 캠페인 총 지표 산출
- 플랫폼별 성과 비교
- 시각화 차트

### 5. PDF 리포트 생성
- 광고주 제출용 템플릿
- 깔끔하고 세련된 디자인
- 즉시 다운로드 가능

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| 크롤링 | Python + Selenium/Playwright |
| 웹 UI | Streamlit |
| PDF 생성 | reportlab 또는 weasyprint |
| 시각화 | matplotlib/plotly |
| 인증 | 기본 인증 (내부 사용자용) |

---

## 주의사항

### 샤오홍슈 QR 인증
- 완전 자동화 불가
- 브라우저 띄워서 사용자가 QR 인증하는 방식
- 인증 후 세션 유지하여 크롤링 진행

### 인스타/페이스북 방어
- 봇 탐지 우회 처리 필요
- Rate limiting 고려
- 필요시 Playwright stealth 모드 사용

### 디자인
- 깔끔하고 세련된 PDF 템플릿 요구
- 기본 템플릿으로 제공 예정
